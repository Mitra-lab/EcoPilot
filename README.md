# EcoPilot

AI-powered sustainability guidance for everyday actions.

Author: Mitra A (Senior Software Engineer)

## Problem Statement

Many carbon footprint tools stop at awareness by calculating emissions and displaying reports.

Users often struggle to:

- Take meaningful action
- Build sustainable habits
- Measure improvement over time

EcoPilot bridges this gap through personalized recommendations, challenge-based engagement, AI-assisted verification, and impact tracking.

---

## Features

### Carbon Assessment

Generate a personalized sustainability score.

### AI Sustainability Coach

Receive customized recommendations.

### Weekly Challenges

Track actionable sustainability goals.

### AI Verification

Verify challenge completion using text or image evidence.

### Green Points

Earn rewards for sustainable actions.

### Impact Dashboard

Track carbon reduction progress.
